Oh Johnny

Make use of *duino right in your browser.

Download .zip Download .tar.gz View on GitHub

So what is it with this Johnny?

Oh Johnny enables you to watch when sensors are being connected to your *duino board and what kind of signal they produce. Oh Johnny makes use of the Johnny-Five library to read sensor data which in turn relies upon a Firmata prototcol implementation for Node.js.

How to use it

Since Oh Johnny connects the physical world with the digital, a server needs to be started on the machine to which an *duino with the Standard Firmata is connected, before any browser client may read out and display its signals:

$ npm install # needed only once to install dependencies
$ node server.js

After that, you may point your browser to http://localhost:3000 and watch your Johnny in action. (This is where the project name starts to sound creepyâ€”Ed.)

Current State

Right now, Oh Johnny doesn't really read signals from your board, mainly because filtering out analog noises and detecting pin connects seems to be a project in itself. We decided to focus on signal display first, this is why your Johnny seems to be firing events on itself.

Help welcome!

Anybody with a grasp of knowledge on electronics and/or Node.js programming is welcome to help with the analog noise/connection detection issues preventing this project from unleashing its full potential.
